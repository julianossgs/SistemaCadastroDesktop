SISTEMA DE CADASTRO

Sistema de cadastro desenvolvido na linguagem C# e banco de dados SqlServer usando o .Net Framework 4.7.2 para desktop

O sistema gerencia um cadastro de cliente e os produtos que foram vendidos a este cliente

Possui também a opção de impressão para os Clientes e Produtos cadastrados


O sistema possui um cadastro de clientes com
- Nome
- Sobrenome
- Email
- Data de cadastro
- Status do cliente

Possui também um cadastro de Produtos com
- Nome
- Valor
- Disponível
- Cliente
